# my-pens-vue

Here are my javascript sketches, wrapped in a vue app. The repository exists for the purpose of demonstrating my code.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
